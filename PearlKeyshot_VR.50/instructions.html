Instructions for sharing KeyShotVR

Below is your KeyShotVR. To view the KeyShotVR by itself open the PearlKeyshot_VR.50.html file in the PearlKeyshot_VR.50 folder.

Uploading to your web site:

  • Inside your PearlKeyshot_VR.50 folder, upload the "files" folder and PearlKeyshot_VR.50.html file to a publicly accessible server.
  • In your webpage where you would like your KeyShotVR to be viewed from, create an <iframe> tag in the <body> section of the html file and update the <src> attribute to reflect the location of the VR on the webserver. Example: <iframe src="PearlKeyshot_VR.50.html" width="1287" height="720" frameborder="0"></iframe>
  • The PearlKeyshot_VR.50.html file must be kept in the same directory as the "files" folder.

If you are sharing via email. You will need to send the entire zipped PearlKeyshot_VR.50 folder to the end recipient.
